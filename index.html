cors-poc

Binance PoC — Proof of Concept files for CORS and API checks

About

This repository contains scripts and sample output used to test CORS headers and API endpoints for web3api.binance.com and related hosts. Use these files only with a test account you own and follow the program rules.

How to run scripts locally (safe, read‑only)

  1. Clone this repo or download the files.
  2. Open terminal in the repository folder and make scripts executable: 
    chmod +x curl_requests/*.sh
  3. Run the preflight (OPTIONS) check and save results: 
    ./curl_requests/preflight_options.sh > results/preflight_options_output.txt 2>&1
  4. Run GET headers check: 
    ./curl_requests/get_requests.sh > results/get_requests_output.txt 2>&1
  5. Sanitize headers before sharing (remove cookies/tokens): see README or run: 
    sed -E '/Set-Cookie:/Id; /Authorization:/Id' results/get__api_v1_user_profile_headers.txt > results/get__api_v1_user_profile_headers.sanitized.txt

Important: Always use a test account you own (no funds) and avoid any write operations (no transfers, trades, deposits, withdrawals).
